# Kenoma Studio Game Library

## Overview

This project is a web-based game library built as a single HTML file. It functions as a simple game launcher interface where multiple games, prototypes, or interactive projects are displayed in a catalog-style layout.

The entire project is contained within one HTML document, including structure, styling, and scripting.

---

## Description

The goal of this project is to simulate a game library or launcher interface similar to a digital storefront. Each game is presented as a card with interactive actions such as play, install, or coming soon.

This is an experimental and evolving project focused on design, interactivity, and user experience in a single-file web environment.

---

## Features

- Single-page game library interface
- Game cards with individual sections
- Interactive buttons (Play / Install / Coming Soon)
- Embedded animations and visual effects
- Custom UI inspired by game launchers

---

## Technologies Used

- HTML5
- CSS3 (embedded in HTML)
- JavaScript (embedded in HTML)
